## GraphManager  --RU--

## Начальная версия приложения (текущее его состояние!)  v0.1.

# Статус: Приложение находится на этапе раннего прототипа.

# Реализовано:

* Создание блоков на рабочем поле
* Удаление блоков
* Перемещение блоков (Drag & Drop)
* Сохранение проекта в JSON-файл
* Загрузка проекта из файла
* Базовая структура MVVM
* TaskProject, TaskBlock, TaskLink модели
* Рабочий Canvas с панорамированием
* Создание и удаление связей между блоками

# Пока отсутствует:

* Редактирование блоков (название, описание, статус выполнения)
* Проверка пересечений блоков
* Контекстное меню (ПКМ)
* Двойной клик по блоку → окно деталей
* Уведомления интерфейса / визуальная подсветка
* Графически приятный UI (цвета, стили, тени)
* Zoom (масштабирование колесиком мыши)
* Undo / Redo

# Ближайшие планы (Roadmap)

* Добавить окно редактирования блока
* Реализовать двойной клик для открытия деталей
* Возможность менять название и описание
* UI-улучшения для отображения блока
