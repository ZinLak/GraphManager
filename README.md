# GraphManager

### Описание:

GraphManager — визуальное приложение для управления задачами в формате блок-схемы.
Позволяет создавать блоки задач, свободно размещать их на рабочем поле, связывать между собой, редактировать содержимое и сохранять проект.

Приложение используется как инструмент для:

* структурирования идей;
* проектирования процессов;
* построения майнд-карт;
* визуального планирования.

Приложение построено на WPF и использует архитектуру MVVM, что обеспечивает расширяемость, модульность и удобство дальнейшей разработки.

# Текущая версия: v0.3

### Статус: приложение находится на стадии улучшенного MVP.

## Реализовано:
### v0.1
* Создание блоков на рабочем поле
* Удаление блоков
* Перемещение блоков (Drag & Drop)
* Сохранение проекта в JSON-файл
* Загрузка проекта из файла
* Базовая структура MVVM

* Рабочий Canvas с панорамированием

* Уведомления интерфейса
* Исправлены недочеты с коллизией. Теперь коллизия работает стабильно
* Улучшен визуальный вид связей
### v0.3
* Undo / Redo
* Добавлены горячие клавиши
* Кастомные уведомления (вместо MessageBox)

# История изменений

## v0.1 — начальный прототип

* Реализованы базовые операции с блоками
* Добавлено сохранение/загрузка проекта
* Создана первичная архитектура MVVM
* Создана рабочая область Canvas
* Реализована логика проверки пересечений блоков и их запрет на пересечение

## v0.2 - версия MVP

* Изменение содержания блоков (Название и описание), а также переключатель "выполнено"
* Запрет на пересечения. Новый блок будет удален, если не найдет место для создания. Выводится MessageBox с оповещением
* Двойной клик по блоку → окно деталей

## v0.3 - Улучшенная версия MVP

* Создано Undo/Redo и горячие клавиши соответственно ctrl + z, ctrl + y
* Уведомления об ошибке выводятся теперь в нижнем правом углу, вместо messagebox.

Будущие версии будут записываться ниже.
