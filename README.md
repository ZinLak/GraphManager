# GraphManager

### Описание:

GraphManager — это визуальное приложение для управления задачами в формате блок-схемы. Пользователь может создавать задачи в виде блоков, размещать их на рабочем поле, связывать логикой, а также сохранять и загружать проекты. Проект разрабатывается как удобный инструмент для структурирования идей, проектирования, построения личных планов или схем процессов.

Приложение построено на WPF и использует архитектуру MVVM, что позволяет легко расширять функциональность и улучшать интерфейс. В дальнейшем планируется развивать проект до уровня профессионального mind‑map/flowchart инструмента.

Это файл, где фиксируются версии проекта, его текущее состояние и планы на будущее.
Здесь можно удобно отслеживать прогресс разработки.

# Начальная версия приложения (текущее его состояние!)  v0.1.

### Статус: Приложение находится на этапе раннего прототипа.

## Реализовано:

* Создание блоков на рабочем поле
* Удаление блоков
* Перемещение блоков (Drag & Drop)
* Сохранение проекта в JSON-файл
* Загрузка проекта из файла
* Базовая структура MVVM
* TaskProject, TaskBlock, TaskLink модели
* Рабочий Canvas с панорамированием
* Создание и удаление связей между блоками


**В новой версии:**
* Проверка пересечений блоков
* Коллизия между блоками
* Редактирование блоков (название, описание, статус выполнения)
* Двойной клик по блоку → окно деталей
* Уведомления интерфейса

## Необходимо доработать:

Иногда блоки не возвращаются на оригинальную позицию, если не находят место, а остаются поверх других. 

## Пока отсутствует:

* Визуальная подсветка
* Графически приятный UI (цвета, стили, тени)
* Zoom (масштабирование колесиком мыши)
* Undo / Redo

# История изменений

## v0.1 — начальный прототип

* Реализованы базовые операции с блоками
* Добавлено сохранение/загрузка проекта
* Создана первичная архитектура MVVM
* Создана рабочая область Canvas
* Реализована логика проверки пересечений блоков и их запрет на пересечение

## v0.2 - версия MVP

* Изменение содержания блоков (Название и описание), а также переключатель "выполнено"
* Запрет на пересечения. Новый блок будет удален, если не найдет место для создания. Выводится MessageBox с оповещением
* Двойной клик по блоку → окно деталей

Будущие версии будут записываться ниже.
